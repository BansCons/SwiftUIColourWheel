# SwiftUI Colour Wheel

A colour wheel made all in SwiftUI. 

Uses `CIHueSaturationValueGradient` `CIFilter` to draw it, then uses `RadialGradient` to smooth it out.
Can output in [Red, Green, Blue] or [Hue, Saturation].
